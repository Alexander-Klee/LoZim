# LoZim

Local Zim file search for KRunner 6.
Easily search through downloaded Wikis like Wikipedia or the ArchLinux-Wiki.

## Dependencies

- [ECM](https://github.com/KDE/extra-cmake-modules/tree/master) (kf6)
- libzim
